This project is an Excel-based data analysis of dog and cat breeds, focusing on breed characteristics, popularity, pricing, temperament, and sales trends. The workbook contains structured datasets, pivot tables, and summary reports for exploratory analysis.

📁 Project Overview

The Excel workbook includes multiple sheets containing:

Dog breed dataset
Cat breed dataset
Summary analysis
Pivot table reports

The project demonstrates data organization, filtering, aggregation, and reporting using Microsoft Excel.
